Data File Set: MDFACW02_d1
File Name: Working personnel file for Mound Plant.


Date fields with only 2 digits for the year must be handled with care and caution.
 When importing the data, users must examine the data element as well as the site associated with the data to determine the correct century. 
For example, some birth dates go back to the 1800s, 
whereas employment dates, exposure dates, and vital status dates will be in the 1900s or even the 2000s.
